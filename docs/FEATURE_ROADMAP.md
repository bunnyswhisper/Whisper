# Feature Roadmap — Bunny's Whisper

## Shipped — Launch V1

- E-commerce core (products, cart, checkout)
- Paymob + COD
- Admin dashboard (orders, inventory, products, analytics)
- Bunny Points loyalty
- Event QR discounts
- Reviews system
- Abandoned cart recovery
- PWA + luxury dark UI

## Shipped — Launch V1.1

- Customer wishlist (persisted per user)
- First-wishlist reward: 100 Bunny Points once (DB unique guard)
- Admin wishlist analytics
- Admin finance tracker (manual income/expense)

## Planned — V1.2+

| Feature | Notes |
|---------|-------|
| Wishlist share / gift hints | Optional, privacy-safe |
| Finance ↔ orders read-only sync | Do not mutate checkout |
| Low-stock Telegram alerts | Extend notifications module |
| Customer notification preferences | Email/SMS opt-in |
| Multi-currency display | EGP remains default |

## Explicitly out of scope (for now)

- Marketplace / multi-vendor
- Subscription billing
- Native mobile apps (PWA only)
- Deployment architecture migration
